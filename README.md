# Batuylabatma.com

Batuylabatma'nın kişisel markası için Hugo (PaperMod teması) ile oluşturulmuş, minimalist ve hacker estetiğine sahip web sitesi.

## Özellikler
- **PaperMod Teması:** Modern ve hızlı.
- **Karanlık Mod:** Varsayılan olarak aktif.
- **Profil Modu:** Sosyal medya bağlantıları ve biyografi içeren ana sayfa.
- **Saha Notları:** Siber güvenlik, biyohack ve yaşam optimizasyonu içerikleri.
- **Otomatik Dağıtım:** GitHub Actions ile her push sonrası otomatik canlıya alınma.

## Yerel Geliştirme
Siteyi yerel ortamda çalıştırmak için:
1. Hugo'yu (v0.146.0+) yükleyin.
2. `hugo server -D` komutunu çalıştırın.
